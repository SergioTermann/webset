# SmileBright 牙科诊所网站

## 项目概述

SmileBright是一个现代化的牙科诊所网站，专为提供专业的牙科服务信息而设计。网站采用响应式设计，确保在各种设备上都能提供良好的用户体验。

## 功能特点

### 1. 页面结构
- **首页**：展示诊所概况、服务亮点和患者评价
- **关于我们**：介绍诊所历史、团队成员和核心价值观
- **服务项目**：详细介绍各类牙科服务，包括常规牙科、美容牙科、正畸治疗等
- **常见问题**：解答患者常见疑问，提供专业建议
- **联系我们**：提供联系方式、地图位置和在线预约表单

### 2. 技术特点
- 响应式设计，适配各种屏幕尺寸
- 现代化UI界面，提供良好用户体验
- 交互式元素，如FAQ折叠面板、表单验证等
- SVG矢量图形，确保图像清晰度
- 优化的页面加载速度

## 文件结构


## 使用指南

### 本地查看
1. 下载或克隆项目到本地
2. 使用浏览器打开`index.html`文件即可浏览网站
3. 或者打开`preview.html`查看所有页面的导航

### 部署到服务器
1. 将所有文件上传到您的网站服务器
2. 确保保持文件结构不变
3. 通过域名访问网站

## 自定义修改

### 修改内容
- 编辑HTML文件中的文本内容以匹配您的诊所信息
- 更新联系信息、地址和营业时间
- 调整服务项目和价格信息

### 修改样式
- 在`css/style.css`中修改颜色、字体等基本样式
- 在`css/additional.css`中添加或修改特定页面样式

### 添加功能
- 在`js/script.js`中添加或修改JavaScript功能
- 可以集成预约系统、在线聊天等高级功能
